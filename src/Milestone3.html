List of Features



Homepage
     The homepage shows a large calming heart image and an About Us section.
     The About Us section describes the features of the website, which are accessable through the Navbar at the top of the page.

Get Help Now
     When Get Help Now is pressed, a popup appears that gives a helpline phone number and reasuring message.

Informative Resources
     The Informative Resources section gives a dropdown menu to navigate to 5 different pages with links to resources.
     The pages are Anxiety and OCD, Depression, PTSD, Bipolar Disorder, and Schizophrenia.

Help Near You
     Help Near You is a tool to find couseling near your area based on Google Maps API.

Chat Bot
     The Chat Bot will talk with visitors to the site and recommend helpful resources.

Search Bar
     The Search Bar allows users to find helpful resources based on search results.


Home Page


Home Page:
     The homepage is the place where the user starts out on our website. From here they can easily access every other part of the website and they can also learn more about what the website can help them with. The homepage starts out with a calming image so that the users are not overwhelmed with a large section of text as soon as they get to the site. Underneath the picture, we have a section of text that informs the users of the various parts of the website so that they can navigate to where they need to go easily.




Get Help Now


Get Help Now:
     The get help now alert gives the user a kind message and gives them the phone number that they should call in order to get immediate help from a trained professional. All the user has to do in order to get help from any part of the website is click on the “Get Help Now” tab on the nav bar and a pop up will show up alerting the user. This tab is easy to locate, clear, and simple so that it does not confuse the user and gives then information in the quickest way possible.




Informative Resources


Informative Resources:
     The informative resources section is a dropdown on the navigation bar with 5 common mental disorders: anxiety and OCD, depression, PTSD, bipolar disorder, and Schizophrenia. This allows the user to learn more about what they are going through.
     These pages have three sections: Support, Financial help, and helpful organizations. The Support section provides the user with links to information about the disorder and links to support groups. The Financial help section helps the user look through links that can help them manage the cost of medication and therapy. The organizations sections gives the users links to reputable organizations that can help the user get help with everything that they are going through. All these sections contain links to get help in one place so that the user is not overwhelmed about where to start when they are seeking out help.




Help Near You


Help Near You:
     The help near you section that can be access from the navigation bar lets the user enter in their zip code and then gives them a map and a list links to doctors and therapists near their location. This allows the user to find professional help outside of the website once they decide that they want or need it. Although this feature is not fully functional, the idea of this part is that the user can just provide a location and the website would filter results to give the user a list of places that they can go to in order to get professional help.




Chat Bot


Chat Bot:
     The chat bot section that can be access through the navigation bar allows the user to talk to an AI that can help them work through their feelings and get help. This allows the user to express their feelings while remaining completely anonymous. The chat bot also gives the user links to helpful resources on the website once it understands what they user is going through of if they ask for help with a specific disorder. It provides calming messages to the user to make them feel a little better and encourages the user to get either professional help or resources that can help.




Search Bar


Search Bar:
     The search bar on the navigation bar in the website allows the user to enter in a phrase or keyword and then gives them a list of websites in a way that is not overwhelming.



Useability Specifications


Learnability:
     Users should understand and feel comfortable navigating all the features of the site within 5 minutes of ariving at the site.

Reliability:
     The site should be down no more than 30 minutes per week.

Longterm Use:
     Advanced users should be provided tools to interact with the site more efficiently, such as shortcuts.

Initial Expression:
     First time users should feel calm and not overwhelmed when visting the site.

Initial evaluation/testing plan

1. Benchmark tasks
     a. ensure that both user groups (especially the mentally ill) can find and use the “get immediate help” button on navbar quickly and without issues .
     b. Ensure that both user groups can locate information based on the illness they wish to learn about
     c. User can find help near them
     d. User can talk to a Chat Bot to feel better
     e. User can search for links on the internet using the search bar

2. Subjective Questions (all of these are from 1 (bad) to 5 (great))
     a. I found this website useful
     b. I found this website calming
     c. I found this website easy to navigate
     d. I felt that using this website was intuitive
     e. I felt intimidated by this website
     f. We might also use questionnaires like the Questionnaire for User Interface Satisfaction(QUIS) or the “System usability scale” questionnaire.

3. Interview questions - Before
     a. Have you ever used online mental health services before? If so, how was your experience with it?
     b. Would you be comfortable using a website to get help if your identity remains anonymous?

4. Interview questions - After
     a. Any comments on the website? Just tell us your first impression.
     b. Was there anything that stuck out to you as particularly well made or internesting.
     c. Was there anything that stuck out to you as poorly designed or hard to use.
     d. (If they had said they had used a mental health web service before) How does our website compare to the one you had used previously? Was there a feature that you feel is important that our website doesn’t have?

5. Other evaluation techniques
     a. We are having each user explore our entire website, and we only have one design (the within approach). Instead of using this approach, we could have made multiple designs and have different user groups test out these different designs (the between approach). Due to time and development constraints, as well as limited availability of users to test the site, we found it appropriate to use the within approach.
